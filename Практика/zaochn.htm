МЕТОДИЧЕСКИЕ РЕКОМЕНДАЦИИ ДЛЯ РЕШЕНИЯ ЗАДАЧ ИЗ КОНТРОЛЬНЫХ РАБОТ ДЛЯ ЗАОЧНИКОВ

Задача 1. Найти близкую к минимальной раскраску и минимальную раскраску (через решение задачи о кратчайшем покрытии) вершин графа, заданного матрицей смежности.

Для получения раскраски графа, близкой к минимальной, удобно воспользоваться методом, описанным в п. 7.2 Теоретического раздела.

Текущая ситуация характеризуется следующими объектами: k – число задействованных цветов; А – множество еще не раскрашенных вершин; В1, В2, … , Вk – совокупность подмножеств множества вершин, такая, что Bi (i = 1, 2, … , k) содержит те и только те вершины из множества А, которые нельзя раскрасить в i-й цвет.

Если имеется вершина, которую нельзя раскрасить ни в один из k цветов, то вводится (k + 1)-й цвет, в который раскрашивается данная вершина. В противном случае выбирается вершина и цвет (например, i-й цвет), раскраска в который приводит к минимальному изменению соответствующего множества Bi.

Минимальная раскраска (точное решение задачи о раскраске) получается сведением ее к задаче о кратчайшем покрытии (гл. 11 Теоретического раздела). Для этого находятся все максимальные независимые множества графа (п. 6.2 Теоретического раздела) и ими покрываются вершины графа. После удаления некоторых вершин из множеств полученной совокупности для того, чтобы эти множества не пересекались, каждое из этих множеств является множеством одноцветных вершин.



Задача 2. В графе, представленном матрицей смежности, найти все максимальные независимые множества и одно наименьшее доминирующее множество.

В п. 6.2 Теоретического раздела описаны два метода получения всех максимальных независимых множеств. Один из них предполагает рассмотрение последовательности подграфов G1, G2, … , Gn заданного графа G с п вершинами, порожденных подмножествами V1, V2, … , Vn, где Vi = {v1, v2, … , vi} (i = 1, 2, … , n).

Нахождение максимальных независимых множеств представляется как многошаговый процесс, на каждом шаге которого множество всех максимальных независимых множеств подграфа Gi преобразуется в такое же множество для подграфа Gi + 1.

Другой из упомянутых методов использует лексикографический перебор независимых множеств.

Нахождение одного наименьшего доминирующего множества графа сводится к решению задачи о кратчайшем покрытии (гл. 11 Теоретического раздела). Для этого матрицу смежности заданного графа надо преобразовать в матрицу, представляющую замкнутые окрестности вершин, т.е. нули на главной диагонали матрицы заменить единицами. Полученную матрицу надо рассматривать как матрицу, фигурирующую в матричной постановке задачи о покрытии, т.е. в ней надо выделить минимум строк, чтобы каждый столбец имел единицу хотя бы в одной из них.



Задача 3. Минимизировать ДНФ методом Блейка-Порецкого. Проверить результат с помощью карты Карно.

Метод Блейка-Порецкого описан в п. 18.2 Теоретического раздела. ДНФ задана в виде троичной матрицы. С помощью операции обобщенного склеивания смежных строк (п. 16.2 Теоретического раздела) получаются вектора, представляющие максимальные интервалы булева пространства. Этими интервалами надо покрыть множество М1, решая при этом задачу о кратчайшем покрытии (гл. 11 Теоретического раздела).

Для сокращения размерности задачи о кратчайшем покрытии необходимо выделить ядро (множество обязательных интервалов) и антиядро (множество интервалов, покрываемое ядром). Ядро целиком включается в решение, а антиядро исключается из рассмотрения. Оставшиеся интервалы используются для покрытия непокрытых элементов множества М1.

Для выделения ядра используется задача о вырожденности троичной матрицы (п. 16.3 Теоретического раздела). Строка ui матрицы, представляющей максимальные интервалы, является избыточной, если ее минор, образованный строками, не ортогональными строке ui и столбцами, где строка ui имеет элемент «–», является вырожденной матрицей. Элемент ядра представляется неизбыточной строкой.

Для проверки результата надо поместить на карте Карно исходное задание и интервалы из полученного решения. Полученные представления должны совпадать.



Задача 4. Удалить из ДНФ избыточные элементарные конъюнкции и избыточные литералы. Проверить результат с помощью карты Карно.

Строки в троичной матрице, представляющей ДНФ, представляют элементарные конъюнкции. Строка ui в такой матрице является избыточной, если минор этой матрицы, образованный строками, не ортогональными строке ui и столбцами, где строка ui имеет элемент «–», является вырожденной матрицей.

Процедура анализа троичной матрицы на вырожденность проводится для каждой строки, и строка, найденная вырожденной, немедленно удаляется. Следует иметь в виду, что строка, избыточная для всей матрицы, может оказаться неизбыточной после удаления некоторой избыточной строки, и ее нельзя удалять.

Литерал, входящий в элементарную конъюнкцию, представляемую строкой ui, представлен в этой строке нулем или единицей. Упомянутый литерал является избыточным, если минор заданной матрицы, образованный столбцами, где строка ui имеет элемент символ «–», и строками, не ортогональными вектору, полученному из ui заменой соответствующего нуля или единицы на противоположное значение, является вырожденной матрицей.

Для проверки результата надо поместить на карте Карно исходное задание и интервалы, представленные строками полученной матрицы. Полученные представления должны совпадать.



Задача 5. Найти минимальную ДНФ для слабо определенной булевой функции, заданной множествами М1 и М0. Проверить результат с помощью карты Карно.

Минимизация слабо определенной функции описана в п. 19.3 Теоретического раздела. Для решения этой задачи надо найти сначала все максимальные интервально поглощаемые множества и затем покрыть этими множествами множество М1, обратившись к задаче о кратчайшем покрытии (гл. 11 Теоретического раздела). Множествам из полученной совокупности соответствуют интервалы, которые после их расширения (п. 19.4 Теоретического раздела) представят решение.

Расширение интервала сводится к задаче о кратчайшем покрытии. При этом надо построить для вектора, представляющего этот интервал, матрицу его различий по отношению к объединению нулевых областей функций, заданных единицами на этом интервале. Находится минимальная совокупность столбцов этой матрицы, где каждая строка имеет единицу хотя бы в одном столбце из этой совокупности. Компоненты рассматриваемого вектора, соответствующие выбранным столбцам, оставляются без изменения. Остальные компоненты приобретают значение «–».

Для проверки результата надо поместить на карте Карно исходное задание и полученные интервалы. Полученные представления должны совпадать.
